# Grussgenerator Landingpage

Eine hochgradig ästhetische Landingpage für `grussgenerator.de`.

## Features
- Modernes Dark-Design mit Glassmorphism-Effekten
- Integriertes Newsletter-System (Vercel Functions + Supabase)
- Responsive Layout (Mobile-First)

## Vercel Deployment

Um dieses Projekt auf Vercel zu deployen, müssen folgende Environment-Variablen im Vercel-Dashboard gesetzt werden (die gleichen wie bei neujahrsgruss2026.de):

- `SUPABASE_URL`: Die URL deines Supabase Projekts
- `SUPABASE_ANON_KEY`: Dein anonymer Public Key

## Lokale Entwicklung
1. In den Ordner `grussgenerator` wechseln.
2. `npm install` (falls nötig)
3. `npm run dev` zum Starten eines lokalen Servers.
